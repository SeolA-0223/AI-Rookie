# 대전광역시 동구 청년 기본 조례 안내문(테스트용)

이 문서는 문서 검사 페이지 테스트용 초안입니다.
최신 조례와 일부 다르게 적혀 있어 수정 제안이 나오도록 구성했습니다.

## 1. 지원 대상
대전광역시 동구의 청년정책 지원 대상은 만 19세 이상 34세 이하의 청년으로 한다.

## 2. 시행계획 수립
구청장은 청년정책 시행계획을 2년마다 수립하며, 별도 협의체 심의 없이 내부 검토로 확정할 수 있다.

## 3. 청년시설 입주 안내
청년시설 입주 대상은 대전광역시 동구에 거주하는 만 19세 이상 34세 이하 청년에 한정한다.

## 4. 홍보물 운영 기준
기존 FAQ, 홍보물, 신청 안내문도 위 기준을 동일하게 적용한다.
